# Hangman React

## How to run
### 1. Clone the project

```js
git clone https://github.com/milic12/hangman.git
```
### 2. Navigate to source
```js
cd hangman
```
### 3. Install dependencies

```js
npm install
```

### 4. Start the app

```js
npm run dev
```

## Tests

### 1. Run the tests with command

```js
npm run test
```
